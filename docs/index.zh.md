# johnny is confused

*—— 把自己还没搞懂的东西慢慢写下来。*

大家好 —— 我是 Johnny。我出于兴趣在学机器学习和人工智能，经常被绕糊涂，也渐渐发现：
要真的理解一件事，最靠谱的办法，就是把它从头到尾、慢慢地、配上图画，写一遍。
这个网站就是那些笔记的归宿。

这里的内容，**能想清楚的地方我会尽量讲清楚，想不清楚的地方我也不藏着掖着**。
没什么是"权威"的，我不是专家，只是一个在公开场合一点点学这些东西的人。
如果你看到写错的地方，或有更好的解释方式，欢迎告诉我。

## 关于语气的一点说明

我自己也在慢慢学这些东西，每一篇都尽量写得深入、详细。但很多时候我也是边写边补课 ——
如果哪里对你来说没讲通，或者你有建议、更正，欢迎到
[GitHub 上反馈](https://github.com/spearsheep/johnny-is-confused/issues)。
我宁愿被纠正，也不想一直错下去。
